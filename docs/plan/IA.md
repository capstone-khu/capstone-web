# IA

캡스톤 디자인의 화면 구조. 로그인 후 **곡 선택 → (혼자 / 협주) → 연주 → 결과** 흐름을 따르며, 지난 연주는 **마이페이지**에서 다시 열람한다.

```
로그인
 └ 곡 선택 (홈)
     ├ 혼자 연주 ─────────────┐
     ├ 협주하기 → 협주 상대 선택 ┤→ 연주 → 결과 → AI 상세 분석
     └ 마이페이지
         ├ 연주 이력 → 결과 (과거 기록 열람) → AI 상세 분석
         └ 집중 반복 레슨 → 연주 (반복 실패 마디)
```

> 모든 화면 전환은 토스풍으로 부드럽게(페이드 + 슬라이드인) 처리한다.

---

## 로그인

- 서비스 타이틀 (CAPSTONE DESIGN)
- 로그인 폼
  - ㄴ 이름 입력
  - ㄴ 비밀번호 입력
- 로그인 버튼

---

## 곡 선택 (홈)

- 상단 헤더 (타이틀 / 우측 '○○ 님, 반가워요' → 마이페이지)
- 안내 카피 (어떤 곡을 연주할까요?)
- 곡 리스트 카드
  - ㄴ 곡 번호 / 곡명
- 모드 선택 바텀시트
  - ㄴ 혼자 연주 / 협주하기
  - ㄴ 협주 안내 문구

---

## 협주 상대 선택

- 상단 헤더 (뒤로 / '협주할 연주 선택')
- 안내 (누구와 함께 연주할까요?)
- 협주 상대 카드 리스트
  - ㄴ 연주자 이름 / 곡명·날짜 / 함께 연주하기
- 음원 로딩 인디케이터

---

## 연주 (메인)

- 권한 요청 화면 (카메라·마이크 안내 / 요청 버튼)
- 카운트인 오버레이 (악보 카드 안에서 박자 카운트)
- 상단 헤더 (뒤로 / 곡명 + 협주 배지 / 마디 카운터 + 일시정지·재개)
- 영상 카드
  - ㄴ 내 카메라 (자세 분석, 자세 경고 시 강조)
  - ㄴ 협주 시 상대 영상은 숨김 — 상대 음원만 재생 (종료 후 좌우 분할 '협주 영상'으로 합성·저장)
- 악보 카드
  - ㄴ 플레이헤드 (악보 고정, 헤드만 이동)
  - ㄴ 누적 마킹 (이전 세션 외곽선 / 이번 세션 채움)
  - ㄴ 페이지 인디케이터 (4마디/페이지)
- 실시간 피드백 영역
  - ㄴ 좌: 지난 연주 피드백 (점선·옅게)
  - ㄴ 우: 현재 연주 피드백 (영역 배지 + 메시지)
- 연주 완료 카드 (홈으로 / 결과 보기)
- 종료 확인 모달
- 집중 반복 레슨 모드 (마이페이지 진입 시 — 한 마디만 10회 반복, 회차 표시, 가이드, 돌아가기·다시 한 번)

---

## 결과

- 상단 헤더 (뒤로 / 결과·곡명 + 협주 배지 / 목록·홈 / AI 상세 분석)
- 누적 마킹 카드
  - ㄴ 범례 (이전 세션 외곽선 / 이번 세션 채움)
  - ㄴ 악보 전체 (6마디 × 2줄 = 1페이지)
  - ㄴ 마디 (클릭 → 상세 모달)
  - ㄴ 페이지네이션 (이전/다음 + 도트)
- 하단 액션 (홈으로·목록으로 / AI 상세 분석)
- 마디 상세 모달
  - ㄴ 큰 악보 + 음표
  - ㄴ 이번 세션 마킹
  - ㄴ 이전 세션 마킹 (참고)

---

## AI 상세 분석

- 분석 로딩 (진입 시 3-dot 인디케이터)
- 상단 헤더 (뒤로 / 'AI 상세 분석 · 곡명' / 홈·목록)
- 디브리핑 카드 (상단 'AI 상세 분석' 띠 + 한 줄 헤드라인 + 코치 코멘트)
- 영역별 탭 (음정 / 박자 / 자세 — 탭 전환, 수준 잘함·보통·아쉬움 + 진단 + 연습)
- 하단 액션 (결과 다시 보기 / 홈·목록)

---

## 마이페이지

- 상단 헤더 (뒤로 / 마이페이지)
- 프로필 카드 (이름 / 로그아웃)
- 집중 반복 레슨 추천 (여러 레슨 반복 실패 마디 → '집중 반복 레슨')
- 연주 이력 리스트
  - ㄴ 곡명·날짜 / '영역별 피드백 확인하기' (→ 결과)
  - ㄴ 영역별 통계 3블록 (음정 / 박자 / 자세 개수)
  - ㄴ 협주 기록은 '협주 영상 보기' (좌우 분할 합성 영상 재생)
