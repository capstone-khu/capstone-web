# 사용자 시나리오

## 시나리오1 : 로그인 → 혼자 연주 → 결과 확인

사용자가 로그인 화면에서 이름과 비밀번호를 입력해 로그인합니다.

곡 선택 화면에서 연주할 곡('반짝 반짝 작은별')을 탭하면 모드 선택 바텀시트가 올라옵니다.

'혼자 연주'를 선택하면 연주 화면으로 이동하고, 카메라·마이크 권한을 요청합니다. 권한을 허용하면 악보 안에서 박자 카운트인(전주)이 진행된 뒤 연주가 시작됩니다.

흐르는 악보의 플레이헤드를 따라 연주하면, 분석 윈도우마다 화면 우측에 실시간 피드백(음정·박자·자세)이 1줄로 뜨고, 악보에는 마킹이 누적됩니다.

곡이 끝나면 완료 카드가 뜨고, '결과 보기'를 탭해 이번 연주의 누적 마킹을 마디 단위로 확인합니다.

---

## 시나리오2 : 협주(듀엣) 연주

곡 선택 화면에서 곡을 탭하고 모드 선택 바텀시트에서 '협주하기'를 선택합니다.

협주 상대 선택 화면으로 이동해 다른 연주자의 녹음 목록 중 하나(예: '김서연')를 탭합니다.

'협주 음원을 불러오고 있어요' 로딩 인디케이터가 잠깐 표시된 뒤 연주 화면으로 진입합니다.

연주 화면에서는 내 카메라와 상대의 녹화 영상이 좌우로 나란히 보이며, 상대 영상은 내 연주 진행과 동기화되어 재생됩니다. 둘이 함께 연주하는 듀엣 흐름으로 진행됩니다.

연주가 끝나면 결과 화면에서 협주 배지와 함께 이번 연주의 마킹을 확인합니다.

---

## 시나리오3 : 연주 중 일시정지 / 종료

연주 도중 사용자가 헤더 우측의 '일시정지' 버튼을 탭하면 악보 진행과 음원이 멈추고, '재개' 버튼으로 다시 이어서 연주할 수 있습니다.

헤더 좌측의 뒤로가기를 탭하면 '연주를 종료할까요?' 모달이 뜹니다. '계속하기'를 누르면 연주로 돌아가고, '종료하기'를 누르면 이번 연주는 저장되지 않고 곡 선택 화면으로 돌아갑니다.

---

## 시나리오4 : 마이페이지에서 약점 확인 → 다시 연주

사용자가 곡 선택 화면 헤더의 '○○ 님, 반가워요'를 탭해 마이페이지로 이동합니다.

상단 코칭 배너에서 그동안 가장 피드백이 많았던 영역(예: 음정)과 '다음엔 음정을 좀 더 신경써 볼까요?' 안내를 확인합니다.

연주 이력 카드에서 각 연주의 영역별 피드백 개수(음정/박자/자세)를 한눈에 보고, 특정 기록의 '영역별 피드백 확인하기'를 탭하면 그 연주의 결과 화면(과거 기록 열람 모드)으로 이동합니다.

결과 화면에서 '다시 연주'를 탭하면 같은 곡으로 다시 연주를 시작합니다.
