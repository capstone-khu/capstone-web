# 협주 상대 선택

## ① 상단 헤더 (App Header)

- **구성**: 좌측 '<' 뒤로가기 + '협주할 연주 선택' 타이틀. 우측 액션 없음.
- **인터랙션**: '<' 탭: 곡 선택(홈)으로 이동.
- **이동 화면**: < → 곡 선택(홈)

## ② 안내 카피 (Heading)

- **구성**: '누구와 함께 연주할까요?' 제목(3xl, Bold) + '「{곡명}」을 함께 연주할 상대를 선택하세요.' 서브 카피(곡명 강조).
- **인터랙션**: 없음. 곡 맥락은 홈에서 이어받음(직접 진입 시 기본 곡으로 폴백).

## ③ 협주 상대 카드 리스트 (Partner List)

- **구성**: 각 카드: 연주자 이름(Bold, xl) + '곡명 · 날짜'(회색) + 우측 '함께 연주하기' 검정 알약 버튼. 해당 곡으로 녹음이 있는 연주자만 표시. (프로필 이미지 없음)
- **인터랙션**: 카드 탭 → 해당 녹음을 협주 상대로 지정하고 음원 로딩 인디케이터(④) 표시 후 연주 진입.
- **상태**: hover 시 배경 강조, 탭 시 살짝 눌림(active:scale). 협주 가능한 녹음이 없으면 '아직 이 곡으로 협주할 수 있는 녹음이 없어요' 빈 상태 카드.
- **이동 화면**: 카드 탭 → (로딩 후) 연주 화면(협주 모드)

## ④ 음원 로딩 인디케이터 (Loading)

- **기능**: 협주 상대 선택 직후 '협주 음원을 불러오고 있어요' + '{이름} 님과 함께 연주할 준비 중이에요' 풀스크린 오버레이. (프로토타입: 실제 로딩 대신 약 1.4초 지연)
- **구성**: 3-dot 바운스 애니메이션 + 안내 문구.
- **인터랙션**: 없음(자동). 완료 시 연주 화면으로 자동 전환.
- **이동 화면**: 자동 → 연주 화면(협주 모드)
