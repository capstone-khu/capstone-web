# 연주 (메인)

> 핵심 인터랙션: 카메라(자세) + 마이크(음정·박자)로 입력받아, **흐르는 악보 위 플레이헤드**를 따라 연주하면 실시간 피드백 1줄과 악보 누적 마킹을 보여준다. 악보는 고정, 플레이헤드만 이동(노래방 방식).

## ① 권한 요청 화면 (Permission)

- **기능**: 연주 분석에 필요한 카메라(자세)·마이크(음정·박자) 권한을 요청. 무엇을 분석하는지(손목·팔꿈치·손가락 커브·어깨·엄지 / 음정·박자) 안내.
- **인터랙션**: '카메라 · 마이크 권한 요청' 버튼 탭 → `getUserMedia` 요청. 허용 시 카운트인(②)으로 진입. '뒤로가기' 탭 → 곡 선택(홈).
- **상태**: 요청 중 '권한 요청 중…' 비활성. 거절 시 안내 박스 노출 + 버튼이 '다시 시도'로.

## ② 카운트인 오버레이 (Count-in)

- **기능**: 노래방처럼 연주 시작 전 박자를 세는 전주. **악보 카드 안에 겹쳐서** 표시(별도 풀스크린 모달 아님).
- **구성**: '곧 연주가 시작됩니다' 안내 + 큰 박자 숫자(1→2→3→4 반복) + 8비트 도트 인디케이터(강박 강조). 메트로놈 사운드 동반.
- **인터랙션**: 없음(자동). 8비트가 끝나면 곡 음원이 시작되고 오버레이가 사라지며 연주 단계로 전환.

## ③ 상단 헤더 (App Header)

- **구성**: 좌측 '<' 뒤로가기 + 곡명. 협주 시 곡명 옆 '협주 · {상대이름}' 배지. 우측: '마디 N/전체' 카운터 + 일시정지/재개 버튼.
- **인터랙션**: '<' 탭(연주 중): 종료 확인 모달(⑦) 오픈(이때 연주 일시정지). '<' 탭(완료 후): 곡 선택으로 바로 이동. '일시정지' 탭: 악보·음원 정지, '재개'로 토글.
- **이동 화면**: < → (모달 경유) 곡 선택(홈)

## ④ 영상 카드 (Camera / Partner Video)

- **구성(솔로)**: 내 카메라 영상(좌우 반전) + 좌상단 'LIVE' 배지 + 우상단 '나' 배지 + 하단 '연주 자세 / 좋은 연주는 바른 자세에서 시작돼요' 푸터.
<!-- - **구성(협주)**: 내 카메라와 상대 녹화 영상을 좌우로 분할. 상대 영상은 placeholder('{이름} 님의 녹화') + 'REC 재생' 배지 + 우상단 상대 이름 + 하단 진행바(내 연주와 동기화). -->
- **인터랙션**: 카메라 미연결 시 '카메라 미연결' 표시.
- **상태**: 자세 영역 지적(posture)이 활성일 때 카드 테두리를 자세 컬러로 강조(글로우).

## ⑤ 악보 카드 (Sheet)

- **구성**: '악보' 헤더 + 1절/2절 라벨 + 페이지 도트. 본문은 **4마디 × 1줄 = 1페이지**(음표를 키우기 위해 한 줄에 4마디). 5선보 위 음표 + 가사 음절. 현재 마디 위로 플레이헤드(헤드만 이동, 악보는 고정).
- **마킹**: 마디 아래 영역별 세로 스택. **이전 세션 = 외곽선**(옅은 가이드), **이번 세션 = 채움**(/60, 실시간 누적). 같은 영역이 겹치면 채움 위에 외곽선을 덧입힘. 영역 컬러: 음정 파랑 / 박자 노랑 / 자세 핑크. GOOD은 표시 없음.
- **인터랙션**: 자동 페이지 전환(현재 마디가 다음 4마디로 넘어가면 페이지 이동).

## ⑥ 실시간 피드백 영역 (Feedback)

- **구성**: 좌/우 50:50 고정. **좌 = 지난 연주 피드백**(점선·옅은 배경, 하단 '지난 연주 피드백이에요!' 캡션), **우 = 현재 연주 피드백**(하단 '현재 연주 피드백이에요!' 캡션). 각 블록 안은 중앙 정렬: 영역 배지(아이콘+이름) + 메시지.
- **톤 구분(3종)**: 일반 지적(영역색 틴트 + 영역 배지) / 칭찬(중립 카드 + 체크 '칭찬' 배지, '잘 하고 있습니다. 계속 유지하세요') / 다중 영역 동시 지적(중립 카드에 영역별 블록 누적).
- **인터랙션**: 없음(분석 윈도우마다 자동 교체). 피드백은 1개 윈도우 동안 유지, 없으면 '다음 마디의 피드백을 분석 중…' / 지난 연주가 없으면 '이 구간은 지난 연주에 받은 피드백이 없어요'를 명시(위치 고정).

## ⑦ 연주 완료 카드 (Finish)

- **구성**: '연주가 끝났습니다' + '이번 연주가 보관함에 자동 저장되었습니다 결과를 확인해주세요' + '홈으로'(아웃라인) / '결과 보기'(검정) 버튼. 곡 종료 시 피드백 영역 아래에 노출.
- **인터랙션**: '결과 보기' 탭 → 카메라 정지 후 결과 화면. '홈으로' 탭 → 곡 선택.
- **이동 화면**: 결과 보기 → 결과 | 홈으로 → 곡 선택(홈)

## ⑧ 종료 확인 모달 (Exit Confirm)

- **기능**: 연주 중 뒤로가기 시 '연주를 종료할까요?' 확인. '지금 종료하면 이번 연주는 저장되지 않고 처음 화면으로 돌아갑니다' 안내.
- **인터랙션**: '계속하기' 탭: 모달 닫고 연주 재개. '종료하기' 탭: 카메라 정지 후 곡 선택으로 이동.
- **이동 화면**: 종료하기 → 곡 선택(홈)

## ⑨ 집중 반복 레슨 모드 (Focus Repeat)

- **진입**: 마이페이지 '집중 반복 레슨' 버튼에서만 진입. 권한 화면을 건너뛰고 준비 로더(3-dot) → 카운트인 → 반복.
- **기능**: 여러 레슨에 걸쳐 반복 실패한 **그 한 마디만** 정해진 횟수(**10회**) 반복(loop) 연습. 일반 연주(곡 전체)와 별개 모드.
- **구성**: 헤더 좌측 '마디 N 집중 반복' / 우측 '반복 N/10회' 카운터 + 일시정지·재개. 악보 카드는 **그 한 마디만 크게** 표시(플레이헤드가 그 마디를 반복). 실시간 피드백 영역 대신 '자주 흔들렸던 마디 N이에요. 천천히, 정확하게 반복해보세요.' 가이드.
- **인터랙션**: '<' 뒤로 → 바로 곡 선택(홈)으로(저장 없음, 종료 확인 모달 없음).
- **완료**: 10회를 마치면 '집중 연습을 마쳤어요' 카드 + '돌아가기'(아웃라인) / '다시 한 번'(검정, 카운트인부터 재시작). 결과 화면으로 가지 않고 저장도 없음.
