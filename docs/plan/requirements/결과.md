# 결과

> 두 가지 진입 맥락: **방금 끝난 연주**(연주 화면에서 진입) / **과거 기록 열람**(마이페이지 이력에서 진입).

## ① 상단 헤더 (App Header)

- **구성**: 좌측 '<' 뒤로가기 + 타이틀('결과 · {곡명}' 또는 과거 기록이면 '{날짜} 연주 · {곡명}'). 협주였던 방금 연주면 '협주 · {상대}' 배지. 우측: '홈으로'/'목록으로'(아웃라인) + 'AI 상세 분석'(검정).
- **인터랙션**: '<'·'홈으로' → 곡 선택(홈) / 과거 기록이면 '목록으로' → 마이페이지. 'AI 상세 분석' 탭: 이번 연주(과거 기록이면 그 기록)의 세션 데이터를 바탕으로 AI 상세 분석 화면으로 이동.
- **이동 화면**: < → 홈/마이페이지 | AI 상세 분석 → AI 상세 분석 화면

## ② 누적 마킹 카드 (Marking Sheet)

- **구성**: 카드 헤더 — 방금 연주면 '이번 연주 누적 마킹' + 자동 저장 안내, 과거 기록이면 '잘못 연주한 부분을 개선해보세요!' + '이 연주에서 피드백 받은 부분입니다…' 안내.
- **범례(③)**: '이전 세션(외곽선)' / '이번 세션(채움)'을 영역 컬러 샘플과 함께 표시.
- **악보**: 전체 24마디를 **6마디 × 2줄 = 1페이지**로. 각 마디는 5선보 + 음표 + 가사 + 누적 마킹(이전=외곽선, 이번=채움).
- **인터랙션**: 마디 탭 → 마디 상세 모달(⑤) 오픈.

## ③ 범례 (Legend)

- **구성**: 음정·박자·자세 컬러 칩 두 줄 — 외곽선(이전 세션) / 채움(이번 세션). 마킹 읽는 법 안내.
- **인터랙션**: 없음.

## ④ 페이지네이션 (Pagination)

- **구성**: '이전' / '다음' ghost 버튼(화살표 아이콘) + 가운데 클릭 가능한 도트 인디케이터. 페이지가 1개면 숨김.
- **인터랙션**: 이전·다음 탭 또는 도트 탭으로 페이지 이동. 양 끝에서 해당 버튼 비활성.

## ⑤ 마디 상세 모달 (Bar Detail)

- **구성**: '마디 N' 타이틀 + 큰 악보(BarView) + 음표 목록 + '이번 세션' 마킹 리스트 + '이전 세션(참고)' 마킹 리스트. 각 마킹: 영역 점 + 영역 이름(+대표 메시지).
- **인터랙션**: 배경·ESC·X로 닫기.
- **상태**: 마킹 없으면 '문제 없음 (GOOD)' / '참고할 이전 마킹 없음' 표시.
- **이동 화면**: 없음(현재 화면 위 모달).

## ⑥ 하단 액션 (Footer Actions)

- **구성**: '홈으로'/'목록으로'(아웃라인, lg) + 'AI 상세 분석'(검정, lg) 2버튼.
- **인터랙션**: 헤더 우측 액션과 동일. 하단에서도 동일 동작 제공.
- **이동 화면**: 홈/마이페이지 | AI 상세 분석 → AI 상세 분석 화면
