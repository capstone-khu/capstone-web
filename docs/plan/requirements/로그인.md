# 로그인

## ① 서비스 타이틀 (Title)

- **구성**: 'CAPSTONE DESIGN' 타이틀(Bold) + '로그인하고 연주를 시작하세요' 서브 카피(회색). 화면 중앙 정렬.
- **인터랙션**: 없음. 안내 표시 영역.

## ② 로그인 폼 (Login Form)

- **구성**: 이름 입력 필드 + 비밀번호 입력 필드. 각 필드 위 라벨(굵은 글씨), 아래 에러 메시지 자리.
- **인터랙션**: 필드 탭 → 키보드 올라옴. 미입력 상태로 제출 시 '이름을 입력해주세요' / '비밀번호를 입력해주세요' 에러 노출 (Zod 검증).
- **상태**: 에러 시 필드 테두리 강조(destructive) + 하단 빨간 안내 텍스트.

## ③ 로그인 버튼 (Submit)

- **구성**: 가로 꽉 찬 검정 CTA 버튼 '로그인' (size lg, 56px).
- **인터랙션**: 탭 → 입력 검증 통과 시 이름으로 로그인 상태 생성 후 직전 진입 화면(없으면 곡 선택)으로 이동.
- **상태**: 제출 중(isSubmitting) 비활성화.
- **이동 화면**: 검증 통과 → 곡 선택(홈) 또는 보호 라우트로 진입하려던 원래 화면.
