# 곡 선택 (홈)

## ① 상단 헤더 (App Header)

- **구성**: 좌측 서비스 타이틀 'CAPSTONE DESIGN', 우측 '○○ 님, 반가워요' 버튼(로그인한 이름, 없으면 '게스트'). 뒤로가기 없음(최상위 화면).
- **인터랙션**: 우측 텍스트 탭: 마이페이지로 이동.
- **이동 화면**: 우측 → 마이페이지

## ② 안내 카피 (Heading)

- **구성**: '어떤 곡을 연주할까요?' 제목(3xl, Bold) + '곡을 선택하면 카메라·마이크 권한을 확인한 뒤 연주를 시작할 수 있어요.' 서브 카피.
- **인터랙션**: 없음.

## ③ 곡 리스트 (Song List)

- **구성**: 카드 안에 '전체 곡 / N곡' 헤더 + 곡 행 리스트. 각 행: 곡 번호(001…, 큰 회색 숫자) + 곡명.
- **인터랙션**: 곡 행 탭 → 모드 선택 바텀시트 오픈.
- **상태**: hover/active 시 행 배경 강조.
- **이동 화면**: 곡 탭 → 모드 선택 시트(④)

## ④ 모드 선택 바텀시트 (Mode Sheet)

- **구성**: 선택한 곡명(타이틀) + '어떻게 연주할까요?' 설명. '혼자 연주' CTA(검정) + '협주하기'(아웃라인) 버튼 + 협주 안내 문구('협주는 다른 연주자의 녹음에 맞춰 듀엣으로 함께 연주합니다'). 우측 상단 X 닫기.
- **인터랙션**: '혼자 연주' 탭 → 솔로 세션 시작 후 연주 화면. '협주하기' 탭 → 곡 맥락을 들고 협주 상대 선택 화면. X·배경·ESC: 시트 닫기.
- **이동 화면**: 혼자 연주 → 연주 화면 | 협주하기 → 협주 상대 선택
