# 마이페이지

## ① 상단 헤더 (App Header)

- **구성**: 좌측 '<' 뒤로가기 + '마이페이지' 타이틀. 우측 **'○○ 님'(없으면 '게스트 님', 좁은 화면에선 숨김) + '로그아웃'(아웃라인, sm) 버튼**. (별도 프로필 카드 없음 — 헤더로 통합)
- **인터랙션**: '<' 탭: 곡 선택(홈)으로 이동. '로그아웃' 탭: 로그인 상태 해제 후 로그인 화면으로 이동(replace).
- **이동 화면**: < → 곡 선택(홈) | 로그아웃 → 로그인

## ② 집중 반복 레슨 추천 (Repeat Coach)

- **기능**: 내 연주 이력에서 같은 마디가 여러 레슨에 걸쳐 반복 실패(슈퍼바이저 폴백 발생)한 경우(세션 간 누적 3회+)를 찾아, 그 마디(들)를 집중 반복 연습하도록 추천. **약점 마디가 여러 개면 모두 표시**(예: '마디 1, 5, 7을 최근 자주 놓쳤어요. 집중해서 반복 연습해볼까요?'). + '집중 반복 레슨'(검정) 버튼.
- **인터랙션**: '집중 반복 레슨' 탭 → 그 **약점 마디(들)** 를 마디 탭으로 전환하며 반복하는 연주 모드(연주 화면 ⑨)로 진입.
- **상태**: 반복 실패 마디가 없으면 표시하지 않음.

## ③ 연주 이력 리스트 (History)

- **구성**: 각 카드 상단 — 좌측 '곡명 · 날짜'(날짜는 '오늘/3일 전' 등 상대 표기), 우측 '영역별 피드백 확인하기 ›'. 카드 하단 — **음정/박자/자세 3블록 통계 그리드**(각 영역 컬러 틴트 배경에 이름 + 개수 큰 숫자). (연주 시간·재생 길이 표시 없음) 목록은 **한 페이지 3개**씩, 3개 초과면 **페이지네이션**(이전/다음 ghost 버튼 + 도트).
- **인터랙션**: 카드 탭 → 해당 기록의 결과 화면(과거 기록 열람 모드)으로 이동. 단독 연주 기록으로 보이도록 모드 초기화 후 recordingId를 결과 화면에 전달.
- **협주 기록**: 협주였던 연주 카드 **안에** '협주 영상 보기' 버튼을 추가로 노출(통계 그리드 아래) → 탭하면 **좌우 분할 합성된 협주 영상**(내 영상 + 상대 영상, 양쪽 음원 합본)을 플레이어 모달로 재생. 솔로 기록엔 없음.
- **상태**: hover 시 배경 강조, 탭 시 살짝 눌림(active:scale).
- **이동 화면**: 카드 탭 → 결과 (과거 기록)
